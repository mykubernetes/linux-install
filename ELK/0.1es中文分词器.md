
ES 索引模块- 停用词
---
有些词在文本中出现的频率非常高，但是对文本所携带的信息基本不产生影响，这样的词我们称之为停用词。
```
中文停用词
https://www.ranks.nl/stopwords/chinese-stopwords

英文停用词
https://www.ranks.nl/stopwords
```

ES 索引模块-中文分词器
```
单字分词：
分词前："我们是中国人"
效果："我" "们" "是" "中" "国" "人"

二分法分词：
比如："我们是中国人"
效果："我们" "们是" "是中" "中国" "国人"

词库分词：
按某种算法构造词，然后去匹配已建好的词库集合，如果匹配到就切分出来成为词语。通常词库分词被认为是最理想的中文分词算法。
```
